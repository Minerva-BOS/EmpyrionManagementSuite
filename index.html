Introducing EMS!

Empyrion Management Suite is a simple, yet robust management tool for Empyrion Galactic Survival. It has an intuitive user interface that will allow you to create, manage, modify, and export custom sectors, playfields, and configurations! These configurations can be published to your single player games, or you can choose to deploy all of the necessary server files necessary to run a dedicated server with SteamCMD. It's really easy and fast to get started. Download the latest stable version by clicking the button above!
